# AI Bot - Todo List

一个基于 React 的 TodoList 管理应用。

## 功能特性

- 添加新任务
- 标记任务完成/未完成
- 删除任务
- 按状态筛选任务（全部/进行中/已完成）
- 统计任务数量

## 技术栈

- React 18.3.1
- JavaScript (ES6+)
- CSS3

## 快速开始

### 安装依赖

\`\`\`bash
cd frontend
npm install
\`\`\`

### 启动开发服务器

\`\`\`bash
npm start
\`\`\`

应用将在 http://localhost:3000 打开。

### 构建生产版本

\`\`\`bash
npm run build
\`\`\`
